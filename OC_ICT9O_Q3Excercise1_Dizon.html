HTML

HyperText Markup Language

It is a filetype that can present text and graphics. It is the contents of the website. It is also the hidden code that allows communication possible worldwide.

CSS

Cascading Style Sheets

It is a programming language designed to make web pages presentable. Serves as the aesthetic purpose of the Website. The colors, symmetry, spacing, positioning and the fonts for the content can be seen here.

Bootstrap

Bootstrap Version 4

Bootstrap is a free front-end framework for faster and easier web development. HTML and CSS properties are involved in bootstrap. It also considered as a toolkit for web development. It can make a very responsive website.
